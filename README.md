# Brain Two

## 项目用途
Brain Two 是我的第二大脑 / 创业工作台 / AI Agent 执行中枢。

它用于沉淀创业项目、Shopify 独立站运营、3D 打印产品管理、AI Agent 执行记录、个人决策系统和每日任务状态。

## 当前阶段
本地 HTML 原型阶段。

## 使用方式
- 当前预期使用方式：直接打开 `brain-two.html`
- 后续可接入本地服务 / API，用于数据同步、AI Agent 调度和项目状态持久化

## 维护规则
- 所有重大变更必须同步更新 `PROJECT_STATE.md` 和 `TODO.md`
- 每次 AI 接手项目前，必须先读取 `README.md`、`CLAUDE.md`、`PROJECT_STATE.md` 和 `TODO.md`
- 不要删除用户已有内容
- 不要进行大规模重构，除非用户明确要求
